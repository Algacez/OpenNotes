# 💫 CI Blog

相信在前面你已经了解了git的一部分使用了,其实git的用处还有很多,例如我们可以利用git的特性建立一个开发流程,可以参考[这篇文章](https://www.cnblogs.com/midworld/p/13608008.html)大致了解一下,当然了要是没有[持续集成](https://zh.wikipedia.org/wiki/持續整合)的话这一切都会变得非常的麻烦,作答本体希望你可以在部署自己的博客时候应用持续集成

## ⭐ 基本要求

- 使用 Hexo / Hugo / Jekyll 等任意静态博客生成工具生成你的博客

- 把你的博客源码上传到Github等Git在线仓库托管平台

- 配置自动集成，使得存放你的博客的源代码的在线仓库更新后，能够自动构建网站并部署

## ❗拓展要求

- 哦我的上帝啊,每次提交都要commit之后还要push实在太麻烦了,能不能在本地git commit时，自动把更新提交到远程仓库呢
- 把博客配置到自己的服务器上面并实现自动集成

## 💡 Hint

- github action,git hook
- 静态web服务端有很多种,想玩apache也可以,不做强制要求
